# CATALOGUE OF IMMERSIVE EXHIBITIONS

It is a catalogue of some pretty well-known and recent immersive exhibitions. Users can add, edit, and delete exhibition entries.

## Technologies Used

- MongoDB, mongoose
- Express
- node.js
- HTML
- CSS
- Adobe Photoshop
- javaScript

- CRUD ACTIONS!
